MedQA Schema Format

Your JSON/JSONL file should contain question objects with the following structure:

{
  "question": "What is the most common cause of community-acquired pneumonia?",
  "options": ["Streptococcus pneumoniae", "Haemophilus influenzae", "Klebsiella", "Legionella"],
  "answer": "Streptococcus pneumoniae",
  "answer_idx": 0,
  "meta_info": "Additional information about the question"
}

For JSONL files, each line should contain a valid JSON object following this schema.
